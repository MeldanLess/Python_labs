1. Опишите класс int.
int — это встроенный в Python неизменяемый тип данных, предназначенный для хранения целых чисел.
После создания экземпляра int его значение нельзя изменить.
Поддерживает все стандартные арифметические операции (сложение, вычитание, умножение, деление, целочисленное деление, остаток от деления, возведение в степень).
Поддерживает битовые операции (например, сдвиги, побитовое И, ИЛИ, исключающее ИЛИ, отрицание).
Может быть создан из других типов данных, таких как строки (при корректном формате) или вещественных чисел (с отбрасыванием дробной части).
Имеет ограниченный набор методов, так как основная функциональность встроена на уровне языка и реализована на C в CPython.
При выполнении операций с другими числовыми типами (например, с float) результатом может быть другой тип (например, float).

2. Как можно реализовать циклы в python?
Цикл for:
Используется для перебора элементов последовательностей (списки, строки, словари, множества, кортежи и т.д.) или других итерируемых объектов.
Цикл while:
Используется, когда выполнение цикла зависит от условия, которое проверяется перед каждой итерацией.

3. Как описывать и вызывать функции в python? Что будет возвращать
описываемая Вами функция по умолчанию?
Функции в Python определяются с помощью ключевого слова def. Далее указывается имя функции, список параметров в круглых скобках и двоеточие, после чего идёт тело функции с отступом.
def my_function(param1, param2):
    result = param1 + param2
    return result
Для вызова функции необходимо написать её имя с соответствующими аргументами в круглых скобках.
sum_result = my_function(3, 4)
print(sum_result)  # Выведет: 7
Если в функции не указан оператор return или он используется без значения, функция по умолчанию возвращает None.
def no_return_function():
    pass

result = no_return_function()
print(result)  # Выведет: None
